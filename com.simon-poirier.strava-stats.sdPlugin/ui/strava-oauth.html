Setup Steps

1. Go to Strava API Settings and create an app:

  • Application Name: Stream Deck Integration
  • Category: Any
  • Website: Any URL
  • Authorization Callback Domain: localhost

2. Copy your Client ID and Client Secret above

3. Copy the Authorization URL below, replace YOUR_CLIENT_ID, and open in browser

4. Authorize the app, then copy the "code" from the redirect URL and paste below
